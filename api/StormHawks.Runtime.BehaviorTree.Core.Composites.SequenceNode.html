    Class SequenceNode

    자식노드들을 왼쪽부터 실행시키는 CompositeNode 만약 실행시킨 자식노드가 Failure를 반환한다면, Failure. 모든 자식노드가 정상적으로 실행되었다면, Success. 자식노드 중 하나라도 실행중이라면, Running.
